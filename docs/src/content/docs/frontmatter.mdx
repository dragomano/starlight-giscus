---
title: Frontmatter
---

By default, the comment block will be displayed on all pages of your documentation.

To disable comments on a specific page, extend the collection schema:

```ts
// src/content.config.ts
import { defineCollection } from 'astro:content'
import { z } from 'astro/zod'
import { docsLoader } from '@astrojs/starlight/loaders'
import { docsSchema } from '@astrojs/starlight/schema'

export const collections = {
  docs: defineCollection({
    loader: docsLoader(),
    schema: docsSchema({
      extend: z.object({
        giscus: z.boolean().optional().default(true),
      }),
    }),
  }),
}
```

and add the property `giscus: false` in the page's frontmatter:

```mdx
// page.mdx
---
title: Some Page
giscus: false
---
```
