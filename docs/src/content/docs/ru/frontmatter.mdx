---
title: Метаданные
---

По умолчанию блок комментариев будет отображаться на всех страницах вашей документации.

Чтобы отключить комментарии на конкретной странице, расширьте схему коллекции:

```ts
// src/content.config.ts
import { defineCollection, z } from 'astro:content'
import { docsLoader } from '@astrojs/starlight/loaders'
import { docsSchema } from '@astrojs/starlight/schema'

export const collections = {
  docs: defineCollection({
    loader: docsLoader(),
    schema: docsSchema({
      extend: z.object({
        giscus: z.boolean().optional().default(true),
      }),
    }),
  }),
}
```

и добавьте свойство `giscus: false` в метаданных страницы:

```mdx
// page.mdx
---
title: Заголовок
giscus: false
---
```
